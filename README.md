High-performance parallel port tester checking system firewall blocks. [Variant 224 using WEB with Matcha theme]

## 🛠️ Usage & Setup

### Requirements
Ensure you have the required runtime environment installed (WEB).

### Execution
Clone the repository and run:
```bash
git clone https://github.com/VarshuAi/go-portscan-v224.git
cd go-portscan-v224
```
Run the master script or compile the executable entry point.
